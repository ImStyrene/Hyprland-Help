# Dependencies
> SwayNC

> Hyprlock

> Hypridle

> Hyprland

> Hyprshot

> Wofi

> Waybar

# Installation
## Arch
```sh
sudo pacman -S hyprlock hypridle hyprland hyprshot wofi waybar
yay -S swaync
```

## Debian/Debian-based
```sh
sudo apt install hyprlock hypridle hyprland hyprshot wofi waybar swaync
```
